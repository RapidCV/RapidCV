## Tech Stack

**Client:** React, TailwindCSS

**Server:** Node, Express


## Installation

```
git clone https://github.com/advaithreddy/RapidCV.git
```

Frontend
```bash
  cd Frontend
  npm install
  npm run dev
```

Backend
```
    cd Backend
```
